# GenAI Prompt Engineering for Customer Review Analysis

## Executive Summary
This project uses structured prompt engineering to analyze customer reviews, classify feedback, and help organizations prioritize operational responses.

## Technical Summary
Generative AI workflows, prompt engineering, text classification, customer experience analysis.

## Approach
- Develop structured prompts
- Analyze customer review text
- Classify feedback into categories
- Identify priority issues for operational response

## Business Impact
Helps organizations quickly understand customer sentiment, identify pain points, and improve service delivery.
